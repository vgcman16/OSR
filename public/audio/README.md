# Audio Clips

Place mission control audio clips in this directory. The soundboard expects the following filenames:

- `mission-start.mp3`
- `mission-event.mp3`
- `mission-success.mp3`
- `mission-failure.mp3`
- `mission-stalemate.mp3`
- `safehouse-alert.mp3`
- `crackdown-shift.mp3`
- `mission-update.mp3`

Files should be short MP3 clips optimized for quick playback in browsers.
